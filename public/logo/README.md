# Logo asset

Drop your logo image here.

**Filename:** `logo.png` (or `logo.svg`)

- **PNG:** Use `logo.png` for a raster image. Recommended height around 32–40px for the sidebar, or use a larger image and the app will scale it.
- **SVG:** Use `logo.svg` for a vector logo. If you use SVG, the app can be updated to reference `logo.svg` instead of `logo.png`.

The logo is shown in the top of the side nav and links to the dashboard.
